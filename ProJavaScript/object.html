Global對象，作為window對象的一部份加以實現。全局的變量和函數都成為window對象的屬性。

獲取Global對象的方法。

var global = function(){ return this;}()

ECMAScript中有兩種屬性：數據屬性和訪問器屬性

數據屬性

  • [[Configurable]]：表示能否通過delete刪除屬性從而重新定義屬性，能否修改屬性的特性，或者能否把屬性修改為訪問器屬性。默認true。

  • [[Enumerable]]：表示能否通過for-in循環返回屬性。默認true。

  • [[Writable]]：表示能否修改屬性的值

  • [[Value]]：包含這個屬性的數據值。

修改默認屬性必須使用Object.defineProperty()
